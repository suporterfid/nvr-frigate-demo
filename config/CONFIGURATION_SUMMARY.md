### Configuration Reference

All go2rtc stream definitions follow patterns from `docs/HOWTO-SIMULATE-FRIGATE-ISSUES.md`:
- Green overlay: colorchannelmixer filter for color shift (matches Option B in HOWTO)
- Frozen frame: `-f lavfi -i color=...` synthetic source producing zero inter-frame difference
- Low bitrate: `-b:v 128k -maxrate 128k -bufsize 64k` constraint on real stream
- Partial obstruction: drawbox filter blacking out left half (960×1080 region) of 1920×1080 frame
- Black screen: `-f lavfi -i color=black` pure synthetic source
